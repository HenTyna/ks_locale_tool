What is this Tool?

The Korean Text Localization Processor is an automated tool that helps developers localize Korean text in TSX/JSX files. It automatically detects Korean text and applies appropriate localization templates (BT or BVT) to make your code ready for internationalization.

Smart Detection

Automatically finds Korean text in JSX elements, attributes, and variable assignments

Template Application

Applies BT or BVT templates to untemplated Korean text

File Download

Download processed files with all Korean text properly templated

How to Use the Tool

1

Upload Your TSX File

Click on the upload area or drag and drop your TSX file. The tool supports files up to 10MB in size.

Tip

Make sure your file contains Korean text (한글) for the tool to process. Files without Korean text will show "No Korean text found" in the results.

2

Choose Template Type

Select between BT Template or BVT Template based on your localization needs:

BT Template

Use for: JSX elements and attributes

// Before
<p>한고어</p>

// After
<p>{bt("W#", "한고어")}</p>

BVT Template

Use for: Variable assignments

// Before
const name = "한고어";

// After
const name = {bvt(name)};
3

Search for Korean Text (Optional)

Click "Find Korean Text" to see what Korean text will be processed without making any changes to your file. This helps you preview the results before applying templates.

Preview Results

The search results will show you the exact Korean text found, its position in the file, and how many elements will be processed.

4

Apply Translation Templates

Click "Apply Translation" to process your file and apply the selected template to all untemplated Korean text.

Important

The tool only processes Korean text that doesn't already have templates. It will skip text that's already wrapped in {bt()} or {bvt()} templates.

5

Download Processed File

After successful processing, click "Download Processed File" to get your updated TSX file with all Korean text properly templated.

File Naming

The downloaded file will be named "processed_[original_filename].tsx" to distinguish it from your original file.

Template Examples

BT Template Examples

// JSX Element Content
// Before:
<h1>제목</h1>
<p>내용입니다</p>

// After:
<h1>{bt("W#", "제목")}</h1>
<p>{bt("W#", "내용입니다")}</p>

// JSX Attributes
// Before:
<input placeholder="입력하세요" />
<button title="클릭하세요">버튼</button>

// After:
<input placeholder={{bt("W#", "입력하세요")}} />
<button title={{bt("W#", "클릭하세요")}}>{bt("W#", "버튼")}</button>

BVT Template Examples (Holding)

// Variable Assignments
// Before:
const message = "안녕하세요";
const title = "제목";

// After:
const message = {bvt(message)};
const title = {bvt(title)};

// Object Properties
// Before:
const config = {
  name: "설정",
  description: "설명입니다"
};

// After:
const config = {
  name: {bvt(name)},
  description: {bvt(description)}
};

Important Notes

Current Limitations

  • BVT Template is temporarily disabled - only BT Template is available
  • Only processes Korean text (한글) - other languages are not supported
  • Maximum file size is 10MB
  • Only supports TSX file format

Best Practices

  • Always preview results using "Find Korean Text" before applying templates
  • Keep backups of your original files
  • Test the processed file in your development environment
  • Use BT Template for JSX elements and attributes
  • Use BVT Template for variable assignments (when available)

Need Help?

If you encounter any issues or have questions about using the tool:

  • Check that your file contains Korean text
  • Ensure your file is a valid TSX file
  • Try with a smaller file if you're experiencing timeouts
  • Make sure you have a stable internet connection
